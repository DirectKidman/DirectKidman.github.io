    A問題　A Your First Judge #

    問題概要 #

    文字列 \(s\) が"Hello World!“に等しければYes、そうでなければNoを出力せよ。

    解法 #

    文字列を扱うことのできる型（String型など）を用いることで、等しいかどうかを調べることができる。

    コード例 #

    　Pythonの場合

    1
2
3

    s = input()

print("Yes" if s == "Hello World!" else "No")
